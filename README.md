# Rent Watcher

A rental analytics website that tracks listings and helps users compare rental prices by area, size, and value.

## Current Goal
Build a working minimal viable product that can:
- store rental listings
- display them in a dashboard
- filter by location, price, bedrooms, and square footage
- show basic rent trends

## Planned Features
- listing scraper or data importer
- PostgreSQL database
- React dashboard
- rent-per-square-foot analysis
- deal score
- charts
- deployed demo

## Languages
- Frontend: React + Tailwind
- Backend: FastAPI
- Database: PostgreSQL
- Data: Python scraping / rental datasets
